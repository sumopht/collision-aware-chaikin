🔬 Algorithm Details

Standard Chaikin
for each edge (P₀, P₁):
Q = P₀ + α·(P₁−P₀)
R = P₀ + (1−α)·(P₁−P₀)
repeat n iterations
O(2ⁿ · k) per iteration
Collision-Aware Chaikin
for each new vertex V:
if inside(V, obstacle):
d = penetration depth
n = surface normal
V += d · n · pushStrength
repeat until resolved
O(n · k · maxIter)
